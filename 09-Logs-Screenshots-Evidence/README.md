# Preuves et captures

Répartissez vos captures dans les sous-dossiers suivants :

| Dossier | Contenu attendu |
|---------|------------------|
| `users/` | Profils, réinitialisations, état du compte |
| `groups/` | Appartenances, propriétaires |
| `mfa/` | Paramètres utilisateur MFA, stratégies CA |
| `licensing/` | Attribution de licences dans le portail |
| `errors/` | Messages d’erreur de connexion |

_Nommez les fichiers de façon explicite : `2026-05-02-ituser-mfa-reset.png`._
