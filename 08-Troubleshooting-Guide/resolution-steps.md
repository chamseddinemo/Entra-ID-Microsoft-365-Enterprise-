# Procédure générale de résolution

| Étape | Action |
|-------|--------|
| 1 | Identifier et noter le symptôme exact (texte d’erreur, heure) |
| 2 | Vérifier utilisateur : activé, UPN, licences |
| 3 | Vérifier groupes et propriétaires |
| 4 | Vérifier MFA et politiques CA |
| 5 | Appliquer correctif minimal |
| 6 | Tester avec l’utilisateur |
| 7 | Documenter dans ce dépôt + capture dans `09-Logs-Screenshots-Evidence/` |
