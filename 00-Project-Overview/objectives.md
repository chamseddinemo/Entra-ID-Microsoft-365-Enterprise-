# Objectifs du laboratoire

## Objectifs pédagogiques

| Objectif | Indicateur de réussite |
|----------|-------------------------|
| Configurer un tenant de lab | Tenant créé, domaine primaire noté |
| Gérer le cycle de vie utilisateur | Création / désactivation / suppression documentées |
| Modéliser les accès par groupes | Tableau groupe ↔ membres ↔ applications |
| Gérer les licences M365 | Attribution et impact du retrait connus |
| Sécuriser avec MFA et CA | Au moins une politique CA testée |
| Playbooks helpdesk | 5 scénarios résolus avec captures |
